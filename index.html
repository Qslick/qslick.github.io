My teams robot can been seen competing at the world competition on the top right picture.

 

I am currently acting as one of the programmers for my college Aerial Robotics team. Our current project is developing a semi autonomous drone to accomplish several challenges laid out in our competition. In terms of autonomy we plan to have an array of inferred range sensors in a circular array to detect range on four corners of the drone. The use case for the array is to modify the sensitivity of the user controls depending on how close the drone is to an obstacle. The closer the drone gets to an obstacle the less sensitive the control will be. The array also allows us to have pre-defined adaptive flying sequences if the need proves useful. In addition, we plan to mount a camera on a servo to pivot from it initial angle which is horizontal to the drone to -90 degrees. The goal of the pivot is to angle the camera towards the landing pad in the last challenge and land autonomously using OpenCV.

NativeScript is a specialized version of Angular 2 which I code mobile applications in. NativeScript allows me to code quickly and efferently for the Android and iOS platforms simultaneously. The way this is possible is that NativeScript Complies into native Java code for Android, and Native Objective C code for iOS using only native libraries hens the name NativeScipt.

 

TypeScript is a derivative of JaveScript which is the main Language of Nativescript. Plain Javascript can also be used to code the functionality of an application, In which I use both. In essence I use TypeScript and Javascript to code the core functionality of an applications. Html and Xml to layout elements such as buttons, and Css or Scss to style such elements.

 

Many applications today do not operate in isolation and often require server-side storage, user login, or a combination of other such functionalities. For this I use Google's Firebase to handle my back end. I personally have use Firebase to handle user login and cloud storage.
